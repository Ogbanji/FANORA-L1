# Consensus Mechanism
**Last Updated: December 2025**

Fanora L1 uses a **Snowman-style consensus** adapted for permissioned Avalanche Subnets. Validators are managed via governance, enabling freeze/pause or rotation when required.

## Features:
- Linear chain consensus optimized for high-performance stablecoin transactions.
- Validator rotation controlled by governance proposals.
- Block finality ensured for compliance and security.
- Integration with runtime for transaction validation and block execution.

## Advantages:
- Efficient and secure consensus for regulated financial operations.
- Supports modular extension without breaking the network.
